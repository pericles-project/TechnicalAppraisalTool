What does "Collection" mean?

A collection is a container or folder which contains one or more digital objects. The user is able to add and remove collections from the tool.


What does "Object" mean?

Object here means a complex digital object is made up of interdependent simple digital objects or components (e.g. files, software applications, hardware specifications). In the media domain, it a combination of a digital video file and its playback environment. In science, it is the raw data, scripts and outputs of the experiment.


Explanations of each column in the "Objects" table

Object Name: The name of the digital object.

Creation Date: Date the object was uploaded to the system.

Risk Level: Combined risk proximity and impact measure: 1 – low, 10 – high.

Primary Risk: Type of risk of nearest proximity to the object.

Date for action: The predicted date that action is required.

Confidence: confidence interval of the prediction

Impact: Confidence interval for the date for action.

Delete Selected: Select one or more objects for deletion.


How to create a new collection?

1. Click the "New" button on the sidebar on the left side of the page.

2. Click "Collection".

3. Enter the name of the new collection

4. Click "Create" button.


How to delete an existing collection?

1. Click "Collections" on the sidebar on the left side of the page.

2. Click the bin icon next to the collection to be deleted.

3. Click "Yes" to confirm your selection when the warning message is displayed.


How to create a new digital object?

1. Click the "New" button on the sidebar on the left side of the page.

2. Click "Object".

3. Select the "Collection" for the new object from the dropdown list.

4. Enter the name of the new object.

5. Select the "Accession Date".

6. Select the "Video Player" from the dropdown list.

7. Select the "Operating System" from the dropdown list.

There are two ways to define the video codec and the container format of the object. The first one is uploading the digital video file to extract for automatic detection. To do this:

1. Click "Browse..." button.

2. Select the video file.

3. Click "Upload" button.

After the video file is uploaded and analysed, the analysed video codec and the container format will be displayed.

The second way is to select video codec and container format manually from the dropdown list. To do this:

1. Click "Or select video codec and container format manually".

2. Select the "Video Codec" from the dropdown list.

2. Select the "Container Format" from the dropdown list.

Once all the information for the object is provided, click the "Create" button.


How to delete existing object(s)?

1. Click the checkbox of the object(s) to be deleted.

2. Click "Delete Selected".

3. Click "Yes" to confirm your selection when the warning message is displayed.
