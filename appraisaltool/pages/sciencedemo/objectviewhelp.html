What is the Model Overview?

The model shows the structure of the complex digital object together the names of the components in this particular object.


Explanations of each column in the "Components" table

Type: The type of the component, e.g. Video Player or Image Format.

Name: Name or identifier for the component.

Risk Type: Describes the risk to the component. This can be either Obsolescence or if appropriate Hardware Failure.

Date for action: The predicted date that action is required to replace instances of the component occurring in objects registered in the tool.


Explanations of each column in the "Recovery Options" table

No.: The index number of the recovery option.

Change: The type of the component(s) which need to be changed in each recovery option.

From: The name of the existing component to be changed.

To: The name of the component the recovery option suggests it to be changed to.

New Date for action: The new date for action when each recovery option is applied.

Imact: The impact or cost of the recovery option on a scale from 1-10, where

Apply Change: Click the tick button to apply a particular recovery option.


How to apply a recovery option?

  1. Click the tick button for a specific recovery option.
  2. Double check the existing model and the new model in the pop-up window.
  3. Click the "Confirm" button.
